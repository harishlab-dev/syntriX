# SentriX – Secure IoT Authentication and Attack Detection System

## Overview

SentriX is a lightweight IoT authentication framework designed to secure communication between IoT devices and authentication servers.

The system combines:
- AES-CBC Encryption
- SHA-256 Hashing
- Nonce Validation

to protect against:
- Replay Attacks
- Impersonation Attacks
- Message Tampering

## Features

- Secure device authentication
- Replay attack detection
- Impersonation attack detection
- Streamlit dashboard visualization
- Flask-based authentication server

## Installation
pip install -r requirements.txt

##Run Server

python server.py

##Run Dashboard

streamlit run app.py

##Project Structure

app.py – Streamlit Dashboard
server.py – Authentication Server
device.py – IoT Device Simulator
crypto_utils.py – Encryption Utilities
utils.py – Hashing Utilities
